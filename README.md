## Sobre a disciplina

Estrutura de Dados é a área da computação que estuda formas de organizar e manipular dados para tornar os algoritmos mais eficientes em tempo e memória.

Na disciplina de Estrutura de Dados II, o foco geralmente é aprofundar estruturas mais avançadas e suas aplicações práticas. Entre os tópicos comuns estão:

- Árvores balanceadas
- Árvores N-árias e variações
- Tabelas hash
- Heaps e filas de prioridade
- Entre outros

As implementações deste repositório foram desenvolvidas principalmente em C.

## Como compilar e executar (C)

Certifique-se de ter instalado o `gcc`. Exemplo básico de comando para compilação:

```bash
gcc -std=c11 -O2 -Wall -Wextra main.c -o main 
./main
```
